# Loan Approval Prediction

## Overview

This machine learning project predicts whether a loan application will be approved based on applicant characteristics such as income, credit history, education, marital status, and property area.

## Dataset

The dataset contains 614 loan applications with demographic and financial information.

## Workflow

- Data Cleaning
- Missing Value Imputation
- Exploratory Data Analysis
- Feature Engineering
- One-Hot Encoding
- Logistic Regression
- Model Evaluation

## Features

### Numerical
- Dependents
- ApplicantIncome
- CoapplicantIncome
- LoanAmount
- Loan_Amount_Term

### Categorical
- Gender
- Married
- Education
- Credit_History
- Property_Area
- Self_Employed

## Model

Logistic Regression

## Results

- Training Accuracy: 82.1%
- Test Accuracy: 79.5%
